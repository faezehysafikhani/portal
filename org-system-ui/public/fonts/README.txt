فونت B Nazanin — راهنمای نصب محلی
=====================================

سامانه طوری تنظیم شده که فونت B Nazanin را از همین پوشه (به‌صورت محلی) بخواند،
نه از اینترنت. برای فعال شدن فونت، فایل‌های زیر را داخل همین پوشه (public/fonts) قرار دهید:

  • BNazanin.woff2        (وزن معمولی — ترجیحی، حجم کمتر)
  • BNazanin.ttf          (وزن معمولی — جایگزین)
  • BNazanin-Bold.woff2   (وزن ضخیم — اختیاری)
  • BNazanin-Bold.ttf     (وزن ضخیم — اختیاری)

نکته: نام فایل‌ها باید دقیقاً مطابق موارد بالا باشد (حساس به بزرگ/کوچک بودن حروف).

اگر فقط فایل .ttf را دارید، می‌توانید همان را قرار دهید؛ سامانه کار می‌کند
ولی برای سرعت بارگذاری بهتر است نسخهٔ .woff2 را هم بسازید
(از سایت‌های آفلاین تبدیل فونت یا ابزار fonttools).

تا وقتی این فایل‌ها اضافه نشوند، سامانه به‌صورت خودکار روی فونت جایگزین (Tahoma)
نمایش داده می‌شود و از کار نمی‌افتد.

پیکربندی @font-face در فایل src/index.css انجام شده است.
